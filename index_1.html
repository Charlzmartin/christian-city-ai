No AI experience required. 01 Write a Program Description For: Grant writers, program directors, communications staff COPY THIS PROMPT → You are a nonprofit communications specialist. Write a compelling 3-sentence description of [Christian City's Children's Village / Senior Living / Thrive Graduate Program] for use in a grant application. Emphasize the impact on the people we serve and the need in our community. 📋 Copy Prompt TIP: Swap the program name in brackets for any program you need to describe. Try it for Safe Place, Foster Care, or Healthy Families too. 02 Summarize a Report for Leadership For: COO, program directors, admin staff preparing board materials COPY THIS PROMPT → I am going to paste in some notes and data about [program name or topic]. Please turn this into a clean, professional 1-paragraph summary suitable for a board report. Use clear, confident language. Here are my notes: [paste your notes here] 📋 Copy Prompt TIP: Paste messy bullet points, emails, or raw notes — AI will organize and polish them. Works for program updates, financial summaries, or event recaps. 03 Draft a Donor Follow-Up Message For: Development staff, CEO, anyone managing donor relationships COPY THIS PROMPT → Write a warm, professional follow-up message to a donor named [Name] who pledged $[amount] to support [program] but has not yet fulfilled their pledge. The tone should be grateful and relational, not transactional. Reference the mission of Christian City briefly. Keep it under 150 words. 📋 Copy Prompt TIP: Fill in the brackets with real donor details. Always read and personalize the output before sending — AI gives you the first draft, you make it genuine.
